# Многопоточный логгер

Проект состоит из динамической библиотеки для логирования и консольного приложения. Приложение демонстрирует безопасную запись данных из разных потоков.

## Требования

* Компилятор `g++` с поддержкой стандарта C++17.
* Утилита `make`.
* Операционная система Linux


## Сборка проекта

Проект использует Makefile для автоматизации сборки. Доступные команды:

* `make` — базовая сборка (собирает библиотеку `liblogger.so` и приложение `logger_app`).
* `make clean` — удаление объектных и исполняемых файлов.
* `make rebuild` — полная очистка и пересборка проекта с нуля.

## Запуск приложения

Для запуска укажите имя файла журнала и базовый уровень важности (DEBUG, INFO, WARNING, ERROR).

**Синтаксис:**
```bash
./logger_app <файл_журнала> <уровень>
```

### Использование во время запуска

Ввод формата "УРОВЕНЬ:" текст
Выход: exit, Ctrl+D, Ctrl+C
